# 双人宝藏争夺射击游戏 - 玩法说明

## 游戏概述
这是一款基于局域网联机的双人对抗游戏，玩家将在300秒内展开激烈的宝藏争夺。通过拾取更多宝藏、策略性攻击对手并应对NPC威胁，最终以宝藏数量决定胜负。

## 游戏入口
进入游戏后，主界面将显示两个核心按钮：
- **创建游戏**：作为主机创建局域网游戏房间，等待其他玩家加入
- **加入游戏**：搜索并加入局域网内已创建的游戏房间

## 核心玩法

### 游戏目标
在300秒倒计时结束前，收集比对手更多的宝藏，成为最终胜利者。

### 基本机制
1. **宝藏收集**：场景中散布着可拾取的宝藏，靠近即可自动收集
2. **时间限制**：游戏时长固定为300秒，倒计时结束后结算宝藏数量
3. **胜负判定**：结算时宝藏数量多的玩家获胜，数量相同则判定为平局

### 战斗系统
1. **玩家对抗**：
   - 玩家可以互相攻击，造成伤害
   - 被击败的玩家会进入复活倒计时
   - 复活期间无法收集宝藏，影响最终得分

2. **NPC威胁**：
   - 场景中存在2个高血量NPC，会主动攻击玩家
   - NPC被击败后不会复活，可永久清除威胁
   - 击败NPC可减少探索过程中的干扰

### 角色状态
- **生命值系统**：每个玩家拥有初始生命值，受到攻击会减少
- **死亡与复活**：生命值归零时角色死亡，等待设定的复活时间后重生
- **复活惩罚**：死亡期间错过的宝藏收集机会可能影响最终胜负

## 操作说明
- **移动**：通过WASD或方向键控制角色移动
- **视角**：鼠标拖动控制视角方向
- **跳跃**：空格键跳跃
- **攻击**：鼠标左键发射弹药攻击其他玩家或NPC
- **拾取**：靠近宝藏自动拾取

## 策略提示
1. 平衡收集宝藏与攻击对手的时间分配
2. 优先清除NPC可减少探索阻碍
3. 合理利用复活时间差扩大优势
4. 关注倒计时，最后时刻的冲刺可能逆转战局

## 胜利条件
倒计时结束时，拥有更多宝藏的玩家获得胜利！
